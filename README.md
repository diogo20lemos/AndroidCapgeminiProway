# AndroidCapgeminiProway

Repositório criado para disponibilizar as atividades referente ao curso de Android ofertado pela Capgemini, ministrado pela Proway.

As atividades estão na pasta "app/src/main/java/br/com/treinamento/**atividadedeacompanhamento**" , dentro desta, conta as pastas atividade02 atividade03 e assim por diante.

Para alternar entre as atividades 1,2,3 etc. Vá no arquivo "app/src/main/java/br/com/treinamento/atividadedeacompanhamento/**MainActivity.kt**" e alterne entre as chamadas das atividades.
